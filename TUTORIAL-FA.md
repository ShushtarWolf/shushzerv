# این باکس — راهنمای یک‌صفحه‌ای پروژه

> **آخرین به‌روزرسانی:** ۹ ژوئن ۲۰۲۶  
> **وضعیت کلی:** نسخهٔ نمایشی (Demo) نزدیک به تکمیل — حدود **۸۵٪** از محدودهٔ v1

---

## پروژه چیست؟

**این باکس (IN BOX S)** یک پلتفرم وب دوزبانه (فارسی پیش‌فرض + انگلیسی) برای **رزرو زمین ورزشی، کلاس گروهی، پیدا کردن همبازی و مدیریت باشگاه** است.

| مورد | توضیح |
|------|--------|
| **الهام** | [alotennis.ir](https://alotennis.ir/) (Alo Play) — ولی **کپی نیست**؛ برند و UI مستقل |
| **هدف v1** | نسخهٔ قابل‌دمو برای دوستان و تست ایده — نه محصول تجاری نهایی |
| **پلتفرم** | وب + PWA (قابل نصب روی موبایل) — در آینده احتمالاً Capacitor برای اپ استور |
| **پرداخت** | فقط «پرداخت در باشگاه» — بدون درگاه واقعی |

## فناوری‌ها

| لایه | ابزار |
|------|-------|
| فرانت‌اند | Nuxt 4، Vue 3، TypeScript، Tailwind CSS |
| بک‌اند | Nuxt Server API (`server/api/**`) |
| دیتابیس | Prisma + SQLite (`prisma/dev.db`) |
| احراز هویت | `nuxt-auth-utils` (ایمیل/رمز — محلی) |
| چندزبانه | `@nuxtjs/i18n` — FA پیش‌فرض RTL، EN در `/en` |
| PWA | `@vite-pwa/nuxt` |
| نقشه | Leaflet (`ClubsMap`) |

### کارهای انجام‌شده (خلاصه)

- دیتابیس کامل با ۲۰+ مدل (کاربر، باشگاه، سانس، رزرو، کلاس، همبازی، چت، XP، نشان، …)
- ۴۳ endpoint در `server/api/`
- ۲۴ صفحه در `app/pages/`
- دادهٔ نمونه با `npm run db:seed`
- طراحی موبایل‌اول با نوار پایین (Tab Bar)
- آیکون‌های اختصاصی ورزش‌ها (`public/icons/sports/`)

### کارهای باقی‌مانده تا «فریز» v1

| اولویت | کار | توضیح |
|--------|-----|--------|
| P0 | **پالایش UI رزرو** | قرص‌های سانس (slot pills) + دکمهٔ چسبان پایین |
| P0 | **بنر نصب PWA** | پیشنهاد «افزودن به صفحهٔ اصلی» پس از اولین رزرو |
| P0 | **آیکون ۵۱۲×۵۱۲ PNG** | برای تجربهٔ نصب بهتر روی موبایل |

## چطور اجرا کنیم؟

```bash
cd ~/Projects/in-box-s
nvm use          # Node 22
npm install
npm run db:push
npm run db:seed
npm run dev
```

| آدرس | محتوا |
|------|--------|
| http://localhost:3000 | فارسی (RTL) |
| http://localhost:3000/en | انگلیسی (LTR) |

**اشتراک با دوستان:** پوش به GitHub + کلون و اجرای دستورات بالا. برای دمو زنده: همان Wi‑Fi با `nuxt dev --host` یا تونل (ngrok).

## جمع‌بندی

این باکس الان یک **دموی کامل و قابل‌لمس** از ایدهٔ «رزرو ورزشی + اجتماعی» است. هستهٔ محصول (رزرو، کلاس، همبازی، نقش‌ها، داشبورد، PWA) **ساخته شده**؛ مرحلهٔ فعلی **پالایش UX، تست و آماده‌سازی برای نمایش** است — نه راه‌اندازی تجاری.

**قدم بعدی پیشنهادی:** تکمیل موارد P0 جدول بالا، تست با حساب‌های دمو، سپس فریز v1 و در صورت نیاز شروع فاز Capacitor.
